# 座號評價查詢系統

一個用於查詢座號評價的網頁應用程式。

## 功能特色

- 🔍 **評價查詢**：輸入座號 (25-34) 查看個人評價
- 📊 **排行榜**：顯示最高和最低評價的前三名
- 🔒 **銳評保護**：需要密碼才能查看銳評內容
- ⏰ **每日限制**：每個座號每天只能查詢一次
- 🔐 **管理員功能**：生成和管理查看密碼

## 評價系統

評價分為四個等級（由高到低）：
- **L** (3分) - 最高評價
- **S** (2分) - 優秀
- **A** (1分) - 良好
- **X** (0分) - 需改進

系統會自動計算所有評價者的平均分數，並轉換為對應的等級。

## 使用方法

### 一般使用者

1. 開啟 `index.html`
2. 輸入座號 (25-34)
3. 點擊「查詢評價」查看評價結果
4. 如需查看銳評，輸入管理員提供的密碼

### 管理員

1. 開啟 `admin.html`
2. 輸入管理員密碼：`RAN+`
3. 生成或設定查看密碼
4. 將密碼分享給需要查看銳評的使用者

## 檔案說明

- `index.html` - 主頁面，用於查詢評價
- `admin.html` - 管理員頁面，用於管理密碼
- `data.csv` - 評價資料檔案

## 資料格式

`data.csv` 格式說明：
- 第一行：標題行（評價者, 25號, 26號, ...）
- 第 2-11 行：10 位評價者的評分 (0=X, 1=A, 2=S, 3=L)
- 第 12 行：銳評內容

## 技術特點

- 純 HTML/CSS/JavaScript，無需伺服器
- 使用 localStorage 儲存密碼和查詢記錄
- 響應式設計，支援行動裝置
- 美觀的漸層色彩設計

## 本地運行

直接用瀏覽器開啟 `index.html` 即可使用。

如需在本地伺服器運行：
```bash
# Python 3
python -m http.server 8000

# 或使用 Node.js http-server
npx http-server
```

然後在瀏覽器中訪問 `http://localhost:8000`
